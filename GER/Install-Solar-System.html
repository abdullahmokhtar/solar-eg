Sieben Schritte zu Ihrem Sonnensystem

Photovoltaik- und Solarwärmeanlagen sind etabliert, bewährt und zukunftsweisende Technologien zur Strom- und Wärmeerzeugung Solarenergie.
Das müssen Sie wissen
Eine Solaranlage mit ca. 20 m2 Dachfläche deckt 80 % des Jahresjahres ab Strombedarf einer durchschnittlichen Einfamilienhausfamilie vier. Vier m2 Sonnenkollektoren kombiniert mit einem Wassertank decken mehr als ab Die Hälfte des jährlichen Warmwasserbedarfs einer Person wird von einer vierköpfigen Familie benötigt.
Das können Sie tun
Die Installation von Photovoltaik- und Solarheizanlagen ist einfach, wird durch Fördermittel gefördert und leistet einen wichtigen Beitrag zur Energiewirtschaft Zukunft. Mit den folgenden Schritten gelangen Sie schnell und einfach dorthin Holen Sie sich Ihr eigenes Sonnensystem.

  1. Ermitteln Sie das Solarpotenzial Ihres Daches oder Ihrer Fassade
  2. Schätzen Sie Ihren eigenen Verbrauchsanteil und steigern Sie Ihren Nutzen Sonnensystem
  3. Überprüfen Sie die Rentabilität Ihrer zukünftigen Solaranlage
  4. Behörden informieren: Solaranlagen muss vor Baubeginn angemeldet werden
  5. Nutzen Sie die Finanzierung
  6. Nehmen Sie die Solaranlage in Betrieb und sorgen Sie für einen langfristigen Betrieb
